# Regional-Sales-Analysis

This project delivers a comprehensive view of regional sales performance across products, channels, and geographies using a full-stack analytics workflow:

🔧 Tools Used: Python (EDA), Power BI (Dashboarding), Excel (Data Cleaning)

📈 Key Insights Uncovered:
- Seasonal revenue and order trends across months  
- Top-performing products and high-margin price bands  
- Channel efficiency and contribution to revenue/profit  
- State-wise and region-wise sales breakdown  
- Customer segmentation by revenue and profitability

🎯 Business Impact:  
Enabled strategic decision-making by identifying growth opportunities, optimizing channel strategies, and highlighting key customer segments.
This project blends technical rigor with business storytelling—turning raw data into actionable insights.
